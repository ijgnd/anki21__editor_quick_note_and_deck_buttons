this is a port to 2.1 of the popular addon Quick note and deck buttons. The configuration/user options for this version are slightly changed because the original authors (ospalh, glutanimate) added support for multiple rows in 2017. This updated version was never published to Ankiweb - just to github. I adjusted this latest version. Use it at your own risk. It works for me. But I don't like how it looks on MacOS. For details see this reddit thread. If you have a solution for this or find another bug let me on reddit (u/ijgnord) or in my github repo. An alternative is the add-on Quick note and deck change menues. # Versions - 2019-10-15 compatible to Keep model of add cards, misc changes (tooltip, focus), ... - 2018-11-10 initial release # Credits The original authors are Steve AW, ospalh, Glutanimate. ospalh has written "no update planned" (in 2019-10-15). All the good parts of this add-on are from the original authors. For compatibility with 2.1 only minimal adjustments were necessary.
